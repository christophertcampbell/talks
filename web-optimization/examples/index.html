Original - Baseline

Typical of an asset-heavy site with no attention to speed optimization

  • Image overload - High-res images, not reponsive, not next-gen, no lazy loading
  • Excessive libraries - Full Bootstrap CSS (unminified), even though we are only using the grid feature and utilities.
  • Excessive web fonts - Importing several web fonts, even though the page does not use them all.
  • YouTube videos - Directly embeded, not lazy loaded
